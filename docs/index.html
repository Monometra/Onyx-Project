How to Install

  1. Go to Settings > Vencord > Plugins and make sure the SpotifyControls plugin is enabled.
  2. Click the settings icon next to the SpotifyControls plugin to open its options.
  3. In the plugin settings, find the "Theme" section and enable the Use Custom Theme toggle.
  4. Paste the @import rule you copied into the text box that appears. The theme will apply instantly.
